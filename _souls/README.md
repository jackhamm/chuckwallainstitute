# Voice profiles (`SOUL.md`)

This directory holds private voice profiles for the Institute's authors. They
are **not published** — the leading underscore on `_souls/` tells Quarto to
ignore the directory at render time.

## Convention

When drafting (or revising) a post attributed to a given author, first read
the relevant `*.md` file in this directory. The profile is the source of
voice; the post is the application of voice to a topic.

The profiles include some private biographical details that should
**never appear on the page**. They exist to color the prose — its rhythm,
what it reaches for, what it refuses to say — without ever being declared.
